## sarコマンド(CPU, memoryの使用量確認)
```
sysstatを使用してCPUの使用率を確認する
yum install sysstat
sar -u
エラーが出る場合、下記ファイルを確認する
/var/log/sa/saXX
※XXは実施する日付（21であれば、sa21）
　空ファイルを格納するとエラーになるため、別サーバーにあるsaXXファイルをリネームして格納すると実施できた
```

# リダイレクト
```
>  ：ファイル出力（上書）
>> ：ファイル出力（追記）

1：標準入力
2：標準出力
3：標準エラー出力

・エラーをリダイレクトしたい場合
2>
※2を記載しないと、ターミナルにエラーが出力される（リダイレクトされない）

・1も同一ファイルに出力したい場合
ls aa 2>&1 tmp.txt

・ターミナルに何も出力したくない場合
/dev/null　<- ゴミ箱みたいなもの
ls aa >/dev/null 2>&1
```
